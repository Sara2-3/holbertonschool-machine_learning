# SQL Introduction

This directory contains my solutions for the SQL Introduction project at Holberton School.

## Project Overview
This project introduces basic SQL concepts and commands including:
- Database management
- Basic SQL queries
- Data manipulation

## Files
- `0-list_databases.sql` - Lists all databases on the MySQL server
- `1-create_database_if_missing.sql` - Creates database if it doesn't exist
- `2-remove_database.sql` - Drops database if it exists

## How to Use
1. Connect to your MySQL server
2. Run each SQL file using: `mysql -u username -p < filename.sql`
