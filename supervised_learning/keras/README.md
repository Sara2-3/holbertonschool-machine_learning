# Machine Learning with TensorFlow & Keras

This repository contains a series of exercises and implementations designed to build a strong foundation in machine learning using TensorFlow and Keras. The tasks progress step by step, covering essential concepts such as model building, optimization, training, validation, and saving/loading models.

The goal of this project is to provide clear, reproducible examples of how modern deep learning workflows are developed and managed in Python.

## About TensorFlow and Keras

TensorFlow is an open-source machine learning framework developed by Google. It provides a flexible ecosystem for building and deploying machine learning models across different platforms.

Keras is a high-level API built on top of TensorFlow. It simplifies the process of creating neural networks by offering user-friendly abstractions while still allowing access to TensorFlowâ€™s powerful backend.

Together, TensorFlow and Keras enable rapid prototyping, scalable training, and production-ready deployment of machine learning models.

## Requirements

- Python 3.9  
- Ubuntu 20.04 LTS  
- NumPy 1.25.2  
- TensorFlow 2.15  
- Pycodestyle 2.11.1  

### File Requirements

All files must:
- Start with `#!/usr/bin/env python3`
- End with a newline
- Follow PEP8 / pycodestyle conventions
- Include documentation for modules, classes, and functions

## Project Tasks

### 0. Sequential
Build a sequential neural network model with Keras.

### 1. Input
Define input layers and manage feature dimensions.

### 2. Optimize
Implement optimization techniques for training.

### 3. One Hot
Work with one-hot encoding for categorical variables.

### 4. Train
Train models on datasets with proper configurations.

### 5. Validate
Validate models to measure generalization performance.

### 6. Early Stopping
Introduce early stopping to prevent overfitting.

### 7. Learning Rate Decay
Apply learning rate schedules for improved convergence.

### 8. Save Only the Best
Save only the best-performing model during training.

### 9. Save and Load Model
Demonstrate saving and loading entire models.

### 10. Save and Load Weights
Work with saving and restoring model weights.

### 11. Save and Load Configuration
Handle saving and loading model configurations.

### 12. Test
Evaluate models on test datasets.

### 13. Predict
Generate predictions using trained models.
